# StreakBeast Skill

This OpenClaw skill provides autonomous habit verification and onchain staking capabilities for the StreakBeast application.

## Overview

The StreakBeast skill enables AI agents to:
- Verify user habit completion through various data sources
- Interact with smart contracts for staking and rewards
- Monitor streak status and trigger notifications
- Execute autonomous transactions based on verification results
